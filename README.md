# myTheme-v3
MikroTik Hotspot Theme v3

This is a free MikroTik Hotspot Template called myTheme-v3.

How to use:

1. Download the .zip file
2. Extract
3. Upload to your MikroTik files
4. Open your Hotspot settings and set the html directory to this folder.
5. Enjoy!
